Получение списка устройств

Получение информации обо всех кондиционерах текущего аккаунта.

Method: GET
Path: /devices

Входные параметры

Параметр Положение Тип Обязательный Описание
Authorization Header String Да Авторизационный ключ.

Выходные параметры

Параметр Тип Описание
devices Object list Список устройств.
Параметры, вложенные в devices
id Int32 Идентификатор устройства.
title String Наименование помещения.
building Object Информация об объекте.
deviceState Object Информация о текущем состоянии кондиционера.
supportedCtrl Object Информация о поддерживаемых режимах работы кондиционера (для использования в методе управления кондиционером).
deviceInfo Object Информация о модели кондиционера.
Параметры, вложенные в building
id Int32 Идентификатор объекта.
title String Наименование объекта.
Параметры, вложенные в deviceState
serialNo String Серийный номер Wi-Fi контроллера
status Enum Статус подключения Wi-Fi контроллера к сети.
Возможные значения:
  • connected - Wi-Fi контроллер подключен;
  • disconnected - Wi-Fi контроллер не подключен.
isOn Enum Состояние работы кондиционера.
Возможные значения:
  • 0 - кондиционер выключен;
  • 1 - кондиционер включен.
curTemp Int32 Текущая температура в помещении.
setTemp Int32 Температура, установленная на кондиционере.
mode Enum Текущий режим работы кондиционера.
Возможные значения:
  • auto - авто-режим;
  • cool - охлождение;
  • dry - осушение;
  • heat - обогрев;
  • vent - вентиляция.
fanFlow Enum Текущий режим работы заслонок обдува.
Возможные значения:
  • off;
  • flow3d;
  • horizontal;
  • vertical.
fanSpeed Enum Текущая скорость вращения вентилятора.
Возможные значения:
  • 0 - авто-режим;
  • 1;
  • 2;
  • 3;
  • 4;
  • 5;
Параметры, вложенные в supportedCtrl
modes Enum Поддерживаемые режимы работы кондиционера.
Возможные значения:
  • auto - авто-режим;
  • cool - охлождение;
  • dry - осушение;
  • heat - обогрев;
  • vent - вентиляция.
minTemp Int32 Минимальная температура, поддерживаемая кондиционером.
maxTemp Int32 Максимальная температура, поддерживаемая кондиционером.
fanSpeedCount Int32 Количество скоростей вентилятора, поддерживаемых кондиционером.
fanFlow Enum Поддерживаемые режимы работы заслонок обдува.
Возможные значения:
  • off;
  • flow3d;
  • horizontal;
  • vertical.
Параметры, вложенные в deviceInfo
brand String Бренд кондиционера.
model String Модель (линейка) кондиционера.
productNum String Продуктовый номер кондиционера.
