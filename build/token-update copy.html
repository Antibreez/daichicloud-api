Обновление токена

Обновление авторизационного ключа.

Method: POST
Path: /auth/token/refresh

Входные параметры

Параметр Положение Тип Обязательный Описание
refresh_token Path String Да Токен для обновления авторизационного ключа.
client_id Path String Да API-ключ. Предоставляется компанией Даичи.

Выходные параметры

Параметр Тип Описание
access_token String Авторизационный ключ.
refresh_token String Токен для обновления авторизационного ключа.
expire_in Int64 Время жизни авторизационного ключа (в сек). Текущее значение 1209600 (~14 дней).

Request:

{
  "refresh_token": "dsfsdfsdfsdfsdfsdfsdfsdfsdf",
  "client_id": "weweqweqcrnhtynetynsf"
}

Response:

{
  "access_token": "dsfsdfsdfsdfsdfsdfsdfsdfsdf",
  "refresh_token": "weweqweqcrnhtynetynsf",
  "expire_in": 864000
}
