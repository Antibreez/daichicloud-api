Получение информации по конкретному устройству

Получение информации о выбранном кондиционере текущего аккаунта.

Method: GET
Path: /devices/{id}

Входные параметры

Параметр Положение Тип Обязательный Описание
Authorization Header String Да Авторизационный ключ.
id Path Int32 Да Идентификатор устройства.
