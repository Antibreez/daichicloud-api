Управление устройством

Задание режима работы выбранному кондиционеру.

Method: POST
Path: /devices/{id}/ctrl

Входные параметры

Параметр Положение Тип Обязательный Описание
Authorization Header String Да Авторизационный ключ.
id Path Int32 Да Идентификатор устройства.
command Path Enum Да Команда управления.
Возможные значения:
  • isOn - включить*выключить кондиционер;
  • setTemp - установка температуры;
  • mode - установка режима работы кондиционера;
  • fanFlow - установка режима работы заслонок обдува;
  • fanSpeed - установка скорости вращения вентилятора.
value Path Enum or Int32 Да Соответствующее допустимое значение для команды управления. См. описание метода Получения списка устройств (объект supportedCtrl).

Выходные параметры

Параметр Тип Описание
done Boolean Результат выполнения операции.

Request:

{
  "command": "setTemp",
  "value": 23
}

Response:

{
    "done": true
}
