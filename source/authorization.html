Авторизация

Получение авторизационного ключа.

Method: POST
Path: /auth

Входные параметры

Параметр Положение Тип Обязательный Описание
email Path String Да Данные аккаунта на daichicloud.ru.
password Path String Да Данные аккаунта на daichicloud.ru.
client_id Path String Да API-ключ. Предоставляется компанией Даичи.

HTTP POST, GET запросы можно отправлять как в обычном формате (Content-Type: application/x-www-form-urlencoded || multipart/form-data), так и в виде json. Далее в примерах для удобства просмотра все входные и выходные параметры запросов описываются в json-формате.

Индикатором выполнения запросов служат стандартный http status code:
2xx - успешное выполнение
4xx - ошибки на стороне клиента
5хх - ошибки на стороне сервера

Пример ответа при ошибке

Response:
Status: 422
{
  "errors": {
      "device": "Нет доступа к устройству"
  },
  "message": "Нет доступа к устройству"
}
